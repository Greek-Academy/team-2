# Calculator app with React + TypeScript + Vite

Live preview: https://calc-sooty-omega.vercel.app/

# How to setup

Use the following command to install necessary packages:
```shell
pnpm install
```

To start local server:
```shell
pnpm run dev
```

To test with jest + testing library:
```shell
pnpm run test
```
